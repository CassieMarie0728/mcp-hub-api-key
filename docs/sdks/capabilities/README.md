# Capabilities
(*capabilities*)

## Overview

Capability and server metadata inspection

### Available Operations

* [listCapabilities](#listcapabilities) - List capabilities of connected MCP servers

## listCapabilities

Returns a summarized view of negotiated capabilities across active connections.

### Example Usage

```typescript
import { McpHubApiTypescript } from "@cassie-marie/mcp-hub-api-typescript";

const mcpHubApiTypescript = new McpHubApiTypescript();

async function run() {
  const result = await mcpHubApiTypescript.capabilities.listCapabilities({
    appSessionCookie: "<YOUR_API_KEY_HERE>",
  });

  console.log(result);
}

run();
```

### Standalone function

The standalone function version of this method:

```typescript
import { McpHubApiTypescriptCore } from "@cassie-marie/mcp-hub-api-typescript/core.js";
import { capabilitiesListCapabilities } from "@cassie-marie/mcp-hub-api-typescript/funcs/capabilitiesListCapabilities.js";

// Use `McpHubApiTypescriptCore` for best tree-shaking performance.
// You can create one instance of it to use across an application.
const mcpHubApiTypescript = new McpHubApiTypescriptCore();

async function run() {
  const res = await capabilitiesListCapabilities(mcpHubApiTypescript, {
    appSessionCookie: "<YOUR_API_KEY_HERE>",
  });
  if (res.ok) {
    const { value: result } = res;
    console.log(result);
  } else {
    console.log("capabilitiesListCapabilities failed:", res.error);
  }
}

run();
```
