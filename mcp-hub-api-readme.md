# MCP Hub API

## Description
The MCP Hub API provides a unified wrapper for discovering MCP servers, creating and managing connections, inspecting capabilities, invoking tools, reading resources, retrieving prompts, streaming events, and handling user/session authentication. This API is designed for both browser-based app sessions and programmatic clients, offering support for cookie-based sessions and bearer-token authentication.

## Features
- **User Authentication**: Supports login/logout with session management via cookies and bearer tokens.
- **Server Management**: Register and list available MCP servers with filtering options by region and status.
- **Connection Management**: Create, initialize, list, and remove connections to MCP servers.
- **Resource Management**: List and read resources, as well as subscribe/unsubscribe to resource updates.
- **Tool Invocation**: Access and invoke tools associated with MCP servers.
- **Event Streaming**: Stream live events and notifications via SSE or WebSocket.
- **Prompts and Completions**: Retrieve prompts and submit completion requests to MCP servers.
- **Health and Capability Checks**: Perform health checks on the system and query server capabilities.

## Basic Usage
### User Authentication
- **Login**:
  ```bash
  curl -X POST https://your-api-url/auth/login \
  -H "Content-Type: application/json" \
  -d '{"username": "your_username", "password": "your_password"}'
  ```

- **Get Current User**:
  ```bash
  curl -X GET https://your-api-url/auth/me \
  -H "Authorization: Bearer your_token"
  ```

### Server Management
- **List Servers**:
  ```bash
  curl -X GET https://your-api-url/servers
  ```
  
- **Register a New Server (Admin only)**:
  ```bash
  curl -X POST https://your-api-url/servers \
  -H "Authorization: Bearer your_token" \
  -H "Content-Type: application/json" \
  -d '{"name": "New Server", "url": "https://newserver.example.com"}'
  ```

### Connection Management
- **Create a New Connection**:
  ```bash
  curl -X POST https://your-api-url/connections \
  -H "Authorization: Bearer your_token" \
  -H "Content-Type: application/json" \
  -d '{"serverId": "your_server_id", "transport": "stdio"}'
  ```
